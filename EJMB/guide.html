HTML Basics

HTML (HyperText Markup Language) is the foundation of every website. Here's the basic structure:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My First Website</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>
  <h1>Hello, World!</h1>
  <p>This is my first web page.</p>
</body>
</html>

Copy this template to start any HTML project. Click on the code to copy it!

Quick Reference

Essential elements and concepts you'll use frequently:

HTML Tags

<h1> - Heading

<p> - Paragraph

<a> - Link

<img> - Image

<div> - Container

CSS Properties

color - Text color

background - Background

margin - Outside spacing

padding - Inside spacing

font-size - Text size

JavaScript

document - Access HTML

getElementById() - Select element

addEventListener() - Handle events

console.log() - Debug output

function() - Create functions

Learning Tips

From my experience as a BSIT student, here are tips that helped me learn web development:

01

Practice Daily

Code every day, even if it's just for 30 minutes. Consistency builds skill faster than marathon sessions.

02

Build Projects

Start with simple pages and gradually increase complexity. Check my Creations page for ideas!

03

Don't Give Up

Everyone struggles at first. When I started, I couldn't center a div. Keep practicing and you'll improve!

Try It Yourself

Ready to practice? Here's a simple exercise:

Exercise: Create a Simple Bio Page

  1. Create a new HTML file
  2. Add your name in an <h1> tag
  3. Write a paragraph about yourself
  4. Add a profile image
  5. Create a list of your hobbies
  6. Add a link to your favorite website
  7. Style it with CSS!

Challenge: Make it responsive for mobile devices!
